# tools/

校验、导入、CI 辅助脚本。**不**放游戏运行时逻辑。

## 规划

| 路径 | 职责 | 落地切片 |
|---|---|---|
| `validate_data/` | 校验 `data/**` JSON、id 唯一、引用完整性 | VS0 可先最小；VS1 必须可用 |
| `import/` | 表格→JSON（可选） | 内容上量后 |
| `ci/` | 本地与 CI 入口脚本 | VS1+ |

## 原则

- 依赖尽量标准库；若加第三方，写进设计包/ADR  
- 校验失败必须以非零退出码退出  
- 不在此目录写 Unity Editor 业务（Editor 工具在 `game/` 内）

当前：仅 README 占位；具体脚本在 VS0/VS1 设计批准后添加。
