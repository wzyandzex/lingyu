# C006–C010 概念卡（旗舰锁定）

> 深度片优先服务 R01；下列保证世界观广度与后续扩展锚点。

---

## C006 焰匣蜥（Yanxialixi）
- 区域：R03 焚脊  
- 澄相：焰序  
- 稀有：U / 体型 S2  
- 概念：胸腔像匣，怒意越高匣缝越亮；战斗越久越危险  
- 结契：助其降温/助其锻造二选一，决定分支进化  
- 定位：进攻曲线代表；教学“情绪即资源”

## C007 潮徽鲀（Chaohuitun）
- 区域：R02 潮徽群岛  
- 澄相：潮汐  
- 稀有：C / 体型 S1  
- 概念：体侧潮徽随潮汐变色，群体游弋如活的潮图  
- 结契：按潮汐节拍投喂/游伴  
- 定位：场地与节奏机制吉祥物

## C008 晶嗅（Jingxiu）
- 区域：R07 幽晶地脉  
- 澄相：澄岩 + 雷纹  
- 稀有：U / 体型 S1  
- 概念：鼻梁晶须可“听”矿物心跳  
- 结契：带它完成一次无破坏的采样  
- 定位：研究线搭档；探索挖矿/避险

## C009 星屑獒（Xingxueao）
- 区域：R09 星坠草原  
- 澄相：澄心  
- 稀有：R / 体型 S2  
- 概念：皮毛嵌细碎星屑，忠诚到会记住主人的恐惧  
- 结契：夜间共守营地，抵御一次恐慌事件  
- 定位：羁绊叙事核；“关系先于数值”的旗手

## C010 律残（Lücan）
- 区域：R10 律城遗环  
- 澄相：墟响 + 澄岩  
- 稀有：E / 体型 S3  
- 概念：大灾变后仍执行失效律法的构装残响  
- 结契：证明“新律”或彻底关闭其核心（叙事分叉）  
- 定位：后期主题BOSS/可伙伴化高难目标
