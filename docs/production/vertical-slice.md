# 垂直切片说明书（Vertical Slice）v0.1

> 这是实现阶段的**唯一主目标**。  
> 未达成切片品质前，禁止铺开全地图、全物种、全系统。

---

## 1. 切片一句话

**在翠语林海的 60–90 分钟里，让玩家完成一次“听见世界—结契伙伴—共同成长—直面失谐”的完整生命体验。**

---

## 2. 为什么这个切片能证明一流

它同时验证：
- 世界氛围是否成立（不是换皮森林）
- 结契是否比砸球更动人且可玩
- 养成/进化是否有仪式感
- 战斗是否可读
- 图志是否像百科而非背包
- 主题“理解 vs 失谐”是否可被感觉到

若此切片平庸，扩大体量也只是放大平庸。

---

## 3. 范围（In Scope）

### 3.1 空间
- 区域：`R01 翠语林海` 可玩子集  
- 子场景：晨雾村边缘、学院前哨、回声小径前段、培育点、失谐洼地、静默之心外观

### 3.2 内容量
| 类型 | 数量 | 说明 |
|---|---|---|
| 精灵 | 8–12 | 强制 C001–C005；其余用区域常见种补齐 |
| 主线节拍 | 1 条短主线 | 见 onboarding + main-story |
| 结契情境 | ≥3 | 静随 / 培育 / 仪式（回声鹿可作半完成钩子） |
| 战斗 | 4–7 场 | 含教学、普通、事件 |
| 进化 | ≥1 | 露滴蛹高光 |
| 世界事件 | ≥1 | 棘影/失谐洼地 |
| 发现链 | ≥2 | 隐藏栖息 + 民谣线索 |
| NPC | 4–8 | 老人、教习、见习研究员、巡林员等 |

### 3.3 系统
必须可玩：
1. 第三人称移动与镜头  
2. 兴趣点交互  
3. 图志（至少外观/生态/战斗部分层）  
4. 观察与结契  
5. 队伍（2–3 栏即可）  
6. 回合战斗（场地/天气最少各 1 种表达）  
7. 羁绊基础反馈  
8. 进化演出  
9. 存档/读档  
10. 基础 UI（HUD、图志、队伍、对话）

### 3.4 明确不做（切片外）
- 多区域旅行
- 完整经济与交易
- PVP
- 联机
- 全属性相克终表精调
- 住宅深度
- C006–C010 全战斗成品（概念引用即可）

---

## 4. 玩家路径（标准路线）

```text
晨雾村 → 雾衔结契 → 图志教学 → 初战
→ 苔行/生态观察 → 露滴蛹结契 → 雨中进化
→ 失谐洼地（棘影） → 回声鹿线索 → 静默之心钩子 → 结算/展望
```

允许轻度自由顺序，但关键旗标要防卡死。

---

## 5. 品质条（Quality Bar）

### 5.1 必须达到
- 美术：统一雾绿风格，关键路径可读
- 音频：区域主题 + 结契成功 + 进化 + 失谐不安
- 手感：移动不腻、交互有反馈、战斗节奏清脆
- 文案：术语一致，图志两文体示例上线
- 演出：进化与回声鹿/失谐至少一处“想截图/想发”

### 5.2 失败即重做
- 精灵像复制粘贴素材
- 结契仍是单一进度条小游戏无理解感
- 战斗只会平 A
- 60 分钟后说不清独特点

---

## 6. 验收标准（DoD）

### 6.1 体验验收
- [ ] 未玩过的人 3 分钟观感能说出独特点（关系/生态/失谐）
- [ ] 12 分钟内第一伙伴建立
- [ ] 30 分钟内出现明确高光
- [ ] 完整跑通核心循环无阻断
- [ ] 至少 1 次“靠理解而非等级”的胜利

### 6.2 内容验收
- [ ] 8–12 精灵均有可辨定位
- [ ] C001/C003/C005（或回声鹿钩子）达到旗舰表现
- [ ] 图志至少 3 层可解锁演示
- [ ] 2 条发现链可完成

### 6.3 工程验收
- [ ] 新电脑可构建运行
- [ ] 存档可靠
- [ ] 无阻断 bug；已知问题列表化
- [ ] 关键配置（精灵/技能/遭遇）数据驱动
- [ ] 基础自动化或手测清单通过

### 6.4 数据验收（建议埋点）
- 到达第一结契时长
- 进化观看完成率
- 战斗平均时长与退出率
- 图志打开次数
- 在哪一步流失

## 9. 风险（切片特有）

| 风险 | 影响 | 缓解 |
|---|---|---|
| 结契过重变QT无趣 | 核心差异失败 | 先做 3 模板，猛打磨反馈 |
| 森林关卡空洞 | 探索无聊 | 兴趣点密度表，每 30–60 秒一事 |
| 进化演出廉价 | 失去高光 | 预算倾斜，宁可少做别的 |
| 战斗教过头 | 节奏拖死 | 硬限教学战场次与台词 |
| 范围膨胀 | 永不完成 | 任何新增先问“是否服务 DoD” |
