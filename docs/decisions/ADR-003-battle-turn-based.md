# ADR-003 战斗基底：回合策略 + 场地/天气 + 轻站位

- 状态：Accepted（用户确认 2026-07-20）
- 日期：2026-07-20
- 决策者：用户确认；制作组执行

## 背景
战斗需服务生态表达、教学可读与羁绊演出，同时控制动作同步成本。

## 决策
采用**回合制策略**，加入**场地/天气**与**前后排轻站位**。半即时不进首发切片。

## 后果
- 正：因果可读、构筑友好、实现路径清晰  
- 负：部分动作玩家可能觉“不够热血”——需用演出与节奏补偿  

## 替代方案
- 半即时（保留研究，不进 VS）  
- 纯动作（与策略图鉴深度目标冲突）
