# ADR-001 产品形态：单机根基 + 异步社交

- 状态：Accepted（用户确认 2026-07-20）
- 日期：2026-07-20
- 决策者：用户确认；制作组执行

## 背景
需要在世界深度、关系养成与长期生命之间取舍。强 MMO 会显著抬高工程与设计复杂度。

## 决策
首发以**可完整单机体验**为根基；社交能力以**异步**形式后置扩展（展示、回放、幽灵助战、研究分享等）。

## 后果
- 正：品质验证快；叙事与生态可控；风险低  
- 负：即时同游社交延后；需避免做成“彻底孤岛”的错觉（可用异步痕迹补世界感）

## 替代方案
- 强在线共享世界（否决于首发）  
- 纯单机无任何社交（可作更窄版本，但不作为默认）
