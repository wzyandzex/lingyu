# ADR-004 结契哲学：共鸣主轨 + 研究辅轨

- 状态：Accepted（用户确认 2026-07-20）
- 日期：2026-07-20
- 决策者：用户确认；制作组执行

## 背景
项目差异化要求“理解后才会跟随”，不能退化为统一捕获道具。

## 决策
- **主轨**：观察 → 试炼/共情 → 共鸣结契（因种而异）  
- **辅轨**：研究与道具提高信息与成功率，不跳过理解  
- 世界解释：结契是互为“锚”，非占有  

## 后果
- 正：独特性强，服务叙事与生态  
- 负：设计与实现成本高于单一捕获；需防 QT 化  

## 缓解
先做 3 个高质量模板（静随、培育、仪式），再扩展类型库。

## 替代方案
- 纯生态研究捕获（偏冷，情感弱）  
- 纯砸球（否决）
