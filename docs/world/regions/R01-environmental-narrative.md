# R01 翠语林海 — 环境叙事物与兴趣点密度

> 状态：v0.1 切片施工用  
> 区域卡：`R01-verdant-echo.md`  
> 目的：把「世界可信」落到**可走路径上的可交互物与时间盒**

---

## 1. 主题在场方式

玩家应在走路中感到：

1. 林海仍美且活（安全生态）  
2. 边缘已有失谐渗入（危机）  
3. 人与灵曾有秩序与守护（青篱/图志痕迹）  
4. 静默之心是远景承诺，不是开场讲解

禁止：只靠 NPC 长对话解释大灾变。

---

## 2. 导览路径时间盒（60–90min 切片总预算中的探索份额）

| 段 | 目标时长 | 玩家应发生 |
|---|---|---|
| 村缘苏醒 | 5–8 min | 移动手感、第一视野、雾衔远见 |
| 苔径学习 | 8–12 min | 交互、目击、图志第一次 |
| 湿谷结契 | 10–15 min | 静随闭环 |
| 林线战斗 | 10–15 min | 教学战 |
| 茧坪进化 | 8–12 min | 羁绊条件 + 进化 |
| 雾隙伦理 | 10–15 min | 棘影事件 |
| 鹿迹与展望 | 8–12 min | 发现链、身份钩子 |
| 冗余/迷路缓冲 | 10 min | 防止全卡死 |

兴趣点**平均间隔**（主路径）：约 **45–90 秒** 有一件「可看/可交互/可记录」的事。  
禁止连续 3 分钟无反馈的纯走路（除非有意做氛围且有音频支撑）。

## 4. 环境叙事物分层

| 层 | 例子 | 玩家成本 |
|---|---|---|
| 必经可读 | 渗斑、铃、茧坪 | 0–1 次交互 |
| 侧步奖励 | 碑、采样旗、护林桩 | 短偏离 |
| 隐藏深度 | 测谐仪、可选民谣残句 | 探索癖 |

## 5. 天气 / 昼夜对兴趣点的影响（切片最小）

| 条件 | 影响 |
|---|---|
| 小雨 | 湿谷结契成功率/稳定性表现更友好（规则见 bonding v0.2） |
| 夜 | 雾衔更活跃；棘影区压力感上升（音频） |
| 晴 | 远眺静默之心更清晰 |

切片不做完整季节；预留字段即可。
