# R01 翠语林海（Verdant Echo）

> 垂直切片主区域

## 名片
- **气质**：湿润、低语、新绿与薄雾；安全中暗藏失谐裂纹
- **一句话**：这里教会你先听世界，再伸手。
- **色彩脚本**：雾绿 / 琥珀光斑 / 浅青澄气 / 夜蓝菌光
- **音乐气质**：木管 + 远处滴水与远鸟；失谐区加入不和谐泛音

## 地理
- 外缘：晨雾村、学院前哨、新手牧野
- 中层：回声小径、鹿桥、观察塔
- 深层：静默之心（主线钩子）、失谐洼地
- 垂直性：树冠层 / 林地面 / 根河层，三层遭遇不同

## 生态
- 常见：雾羽雀型、青苔獭型、露滴虫型
- 少见：回声鹿、菌灯狐
- 危险：失谐侵染的棘藤栖者
- 规则：噪音过大降低部分灵体出现率；雨后大量露栖种

## 文明与 NPC
- 晨雾村：向导、牧养家庭、民谣老人
- 学院前哨：基础结契教习
- 图志见习站：野外记录任务
- 守护团巡林员：失谐警报

## 可玩内容（切片必须覆盖）
1. 新手引导与第一伙伴结契  
2. 观察 → 共鸣结契教学（至少 2 种习性）  
3. 3–4 场教学向战斗  
4. 1 次进化/觉醒演出  
5. 1 个世界事件：失谐洼地净化或封锁  
6. 2 条发现链（隐藏栖息 + 民谣线索）  

## 主线节拍（切片）
1. 抵达 / 世界失谐传闻  
2. 第一伙伴遇险或互相选择  
3. 学会倾听（系统教学）  
4. 林海异变加剧  
5. 静默之心初响（开放后续世界的门）  

## 风险
- 新手区过空或过教条
- 雾气氛围导致可读性差（需路径光引导）
