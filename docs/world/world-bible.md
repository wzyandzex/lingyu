# 世界观圣经（World Bible）v0.1

> 状态：骨架锁定，供系统与精灵设计引用  
> 原则：世界先于系统；所有玩法规则应能在世界内解释

---

## 1. 世界名片

| 项 | 内容 |
|---|---|
| 世界名 | **澄界（Aetherion）** |
| 别称 | 共鸣之陆、双息世界 |
| 核心意象 | 雾海之上的陆块与晶脉，人与精灵以“共鸣”共治生态 |
| 时代名 | **余晖纪**（大灾变后的第三秩序时代） |
| 当前危机 | **失谐扩散**：部分区域共鸣紊乱，精灵暴走、生态链断裂、古代遗迹苏醒 |

### 一句话世界
澄界是一座以“共鸣”为物理与伦理法则的世界——生命通过理解彼此而获得力量；当理解崩坏，世界就会失谐。

## 2. 创世与世界规则

### 2.1 创世简史
1. **初息**：原初之息分化为流动的“澄气”（Aether）与凝定的“界骨”（Terra）。
2. **生灵**：澄气入形，诞下可感意志的精灵；界骨承托，形成可居的陆海。
3. **第一共鸣**：早期人类并非征服者，而是学会倾听澄气的旅人；人与精灵首次结契，文明开始。
4. **繁盛与傲慢**：学院、城邦、牧养帝国兴起；有人试图将共鸣工业化、武器化。
5. **大失谐（大灾变）**：强行榨取澄气导致生态反噬，古代都城沉没，传说级精灵沉睡。
6. **余晖纪**：现存势力以“有限共鸣”重建秩序；但失谐裂纹再次扩大。

### 2.2 可玩的世界法则（设计用）

| 法则 | 世界解释 | 玩法映射 |
|---|---|---|
| 共鸣律 | 理解越深，契约越稳，力量越“成形” | 羁绊、结契、形态稳定 |
| 澄气潮汐 | 澄气随昼夜/季节/地点涨落 | 天气、遭遇表、技能场地 |
| 生态位守恒 | 强行移走关键种会引发连锁失衡 | 区域事件、研究任务 |
| 名相可唤 | 真名/古名接近本质，可影响觉醒 | 图鉴研究、进化道具、传说线 |
| 失谐侵蚀 | 无法被理解的痛苦会污染澄气 | 主线危机、暴走战、净化玩法 |

### 2.3 人与精灵关系哲学
- 结契不是占有，而是**互相成为对方在世界中的“锚”**。
- 精灵可拒绝、可离开、可因失谐而异化。
- 最强力量往往来自共历，而非单纯血脉或等级。
- 图鉴在世界中的身份：既是科学志，也是安魂与理解的仪式文本。

## 4. 势力与组织（7 个）

详细见 `factions/`。摘要：

| 势力 | 立场 | 给玩家的路径 |
|---|---|---|
| **澄语学院** | 官方教育与结契规范 | 训练师 / 学院认证 |
| **图志学会** | 研究、命名、图鉴编纂 | 研究员 / 图鉴编纂者 |
| **青篱守护团** | 生态平衡与反盗猎 | 守护者 / 净化者 |
| **回环竞技盟** | 竞技、规则对战、明星文化 | 竞技冠军路线 |
| **余烬商会** | 中立贸易，灰区契约与稀有材料 | 商人/牧养资源线 |
| **古名教团** | 信仰真名与古代共鸣圣仪 | 传说/觉醒叙事线 |
| **无锚旅团** | 拒绝建制的流浪共鸣者 | 自由探索/非主流结契 |

冲突主轴：  
**效率与控制（学院/商会） vs 生态与敬畏（守护团/教团） vs 自由与边界（旅团）**，竞技盟提供公共舞台，图志学会掌握“命名权”。

## 6. 当前时代冲突（主线发动机）

**失谐扩散**正在从地脉裂口渗出：
- 精灵行为异常、进化畸变
- 区域天气“情绪化”
- 古代真名残响被争夺
- 有势力主张“更强控制”，有势力主张“更深理解”

玩家被卷入的不是“选边打架”那么简单，而是必须回答：  
**在失谐的时代，共鸣究竟意味着责任，还是权力？**

## 7. 主线大纲（四幕）

### 第一幕：听见（新手—翠语林海）
- 主角在失谐边缘的林海觉醒微弱共鸣
- 结识第一伙伴，学会观察与结契
- 发现失谐不是天灾谣言，而是可触的伤口
- 结局钩子：林海深处的“静默之心”开始跳动

### 第二幕：命名（多区域展开）
- 加入/周旋于学院与图志学会
- 以研究与旅行扩大图鉴，理解区域生态
- 势力要你选路：登记、守护、交易或流浪
- 中点反转：某次“正确的控制”导致伙伴险些异化

### 第三幕：失名（危机升级）
- 传说种苏醒，律城遗环开放
- 真名被盗用/武器化
- 玩家队伍经历分裂、重聚与重新理解
- 必须放弃一种“捷径力量”才能保住关系

### 第四幕：锚定（终局）
- 以羁绊与图志真知对抗失谐核心
- 结局分支受身份路线与关键伙伴关系影响
- 世界可被稳定，但不会回到天真的旧秩序——余晖仍在，责任长存

## 9. 对内容生产的约束

1. 新区域必须回答：失谐如何改写本地生态？  
2. 新势力交互必须提供身份路径，不只是任务板。  
3. 新精灵必须能挂到栖息地与文化，不得漂浮无根。  
4. 主线场景优先服务“理解 vs 控制”主题，避免无意义打卡。
