# 势力摘要

> 完整互动表可在系统期扩展。此处保证身份路径可用。

## 1. 澄语学院（Aetherium College）
- **立场**：规范结契，培养可控的共鸣者
- **资源**：教材、认证、训练场
- **冲突**：安全 vs 自由；登记制度惹争议
- **玩家路径**：学院训练师、教习助役

## 2. 图志学会（Codex Society）
- **立场**：命名、分类、保存真知
- **资源**：图鉴权威、研究许可、稀有记录
- **冲突**：知识公开 vs 危险真名封锁
- **玩家路径**：研究员、编纂者

## 3. 青篱守护团（Greenwarden Circle）
- **立场**：生态优先，打击盗猎与失谐扩散
- **资源**：净化技术、巡逻情报
- **冲突**：激进保护 vs 民生开发
- **玩家路径**：守护者、净化专员

## 4. 回环竞技盟（Ring League）
- **立场**：规则下的荣耀与娱乐公共领域
- **资源**：赛事、观众、跨势力中立舞台
- **冲突**：表演化 vs 传统敬意
- **玩家路径**：竞技选手、表演驯养师

## 5. 余烬商会（Ember Exchange）
- **立场**：流通至上，灰区也能谈价钱
- **资源**：材料、稀有蛋、情报
- **冲突**：利润 vs 生态伦理
- **玩家路径**：商旅、牧养经营者

## 6. 古名教团（True-Name Conclave）
- **立场**：真名神圣，共鸣是信仰
- **资源**：仪式、古语、觉醒线索
- **冲突**：神秘主义 vs 现代研究
- **玩家路径**：祭礼旅人、传说追寻者

## 7. 无锚旅团（Unanchored Host）
- **立场**：拒绝被体系定义的流浪共鸣
- **资源**：非正规结契技巧、隐秘路线
- **冲突**：自由 vs 秩序
- **玩家路径**：流浪者、野路驯养者
