# 时间轴（Timeline）

> 摘要版，细节叙事可在主线文档扩展。

## 神话与远古
- **初息**：澄气 / 界骨分化
- **生灵黎明**：原初精灵成形；无文字的共鸣歌谣时代
- **第一共鸣**：人类旅者与林海之灵结契，定居点出现

## 文明上升
- **村落联盟期**：牧养与口述图志传统
- **辉造纪早期**：学院雏形，契约法规萌芽
- **律城时代**：澄气工程、构装守卫、真名研究巅峰

## 崩坏
- **大失谐元年**：律城核心过载，沉为遗环
- **灰雾十年**：迁徙、饥荒、精灵暴走；大量真名失传
- **守护宣誓**：青篱守护团成立，限制高危结契

## 现代（余晖纪）
- **重建百年**：澄语学院体系化；图志学会掌握命名权
- **竞技公共化**：回环竞技盟成为跨势力舞台
- **今岁**：地脉裂口增密，失谐雾团出现在翠语林海边缘
- **玩家时间点**：余晖纪第 128 年，初春

## 对玩法的时间钩子
- 古代遗物 = 进化/觉醒素材
- 失传真名 = 传说任务链
- 沉没技术 = 后期系统解锁（净化、构装协同等）
