# 非目标（Non-goals）

本清单用于防止范围膨胀与灵魂稀释。

## 绝对非目标（首发周期）

1. 强依赖实时 MMO 的大世界社交沙盒  
2. 抽卡/氪金决定精灵强度的商业模型  
3. 无生态逻辑的百种换皮精灵冲数量  
4. 以 PVP 天梯为唯一终局  
5. 移动端点触优先的操作范式（PC 首发）  
6. 开放世界体量优先于单位面积密度  

## 阶段性非目标（垂直切片前不做）

1. 完整 8–12 大区可玩落地  
2. 300+ 精灵全量实现  
3. 复杂交易行与经济闭环  
4. UGC 模组生态  
5. 跨平台账号与云存档全家桶  
6. 完整语音演出  

## 允许研究但默认不进切片

- 半即时战斗变体  
- 住宅装修深度  
- 公会战  
- 赛季通行证  

若要打破本清单，先写 ADR，再改 `product-constitution.md`。
