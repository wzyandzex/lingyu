# 垂直切片 PRD（Vertical Slice Requirements）v1.0

> **状态**：当前唯一可实现需求包（P0）  
> **时长目标**：标准路径 60–90 分钟  
> **区域**：R01 翠语林海可玩子集  
> **服从**：`PRD.md`、`vertical-slice.md`、`AGENTS.md`、ADR-001～008  
> **协作角色**：主策 / 叙事 / 生态 / 系统 / UX / 战斗 / 技术 / 品质  

---

## 1. Problem / Solution（切片层）

### Problem
需要在一个可玩片段里证明澄界不是“又一个抓宠游戏”，而是：

- 理解驱动的结契  
- 会呼吸的生态  
- 有重量的关系与失谐主题  

### Solution
交付 R01 标准路径：

```text
晨雾村 → 雾衔结契 → 图志 → 初战 → 苔行观察
→ 露滴蛹结契与雨中进化 → 棘影失谐事件
→ 回声鹿线索 → 静默之心钩子 → 路径展望
```

## 2. In Scope / Out of Scope

### 2.1 In Scope（必须做）

**空间**
- 晨雾村边缘、学院前哨、回声小径前段、培育点、失谐洼地、静默之心外观（可远景+触发点）

**内容**
- 精灵 8–12：强制 C001–C005；补 3–7 只区域常见种  
- NPC 4–8  
- 结契模板 ≥3：静随、培育/环境、（寻迹帮助或仪式半程）  
- 战斗 4–7 场  
- 进化 ≥1（露滴蛹）  
- 世界事件 ≥1（棘影/失谐洼地）  
- 发现链 ≥2  

**系统**
- F01–F13、F21–F22 的切片子集（见 Master PRD 域表）  
- GameMode：Title（可简）、Exploration、Bonding、Battle、Dialogue、Menu、Cutscene（进化）  
- 存档 ≥1 槽，含版本头  

### 2.2 Out of Scope（禁止做进切片）
- 多区域旅行、完整经济、PVP、联机、异步社交  
- 牧场深度、身份数值体系、全澄相终表精调  
- C006–C010 全战斗成品  
- Addressables 全家桶、DOTS、HDRP  
- 回声鹿完整多日传说（允许钩子/半程仪式）

## 4. 标准玩家路径（分场需求）

### Beat 0 — 启动与进入世界（0–5 min）
**目标**：世界美且不安；学会移动/环视/互动。  
**需求**
- 进入 R01 晨雾村边缘  
- 远景可见失谐雾团  
- 民谣老人短对话（≤6 次点击）点题  
- 教学：移动、镜头、交互键提示  

**验收**
- 新玩家不看说明可完成与老人对话并走到引导点  

### Beat 1 — 雾衔结契（5–12 min）
**目标**：第一高光——它愿意靠近你。  
**需求**
- 触发雾衔观察事件  
- BondingMode：静随模板  
  1. 保持距离跟随巡逻  
  2. 模仿停步侧耳（提示+输入）  
  3. 允许时递出掌心澄气  
- 失败：喷雾遁走，提示厌恶压迫，可重试  
- 成功：加入队伍；Cutscene/短演出  

**验收**
- 成功路径可学会；故意冲撞会失败并给原因  
- E1 计时达标  

### Beat 2 — 图志教学（12–20 min）
**目标**：收集=研究。  
**需求**
- NPC 授予图志簿  
- 目击苔行 → 外观层  
- 完成一次观察动作 → 生态层线索  
- UI 打开图志可读双文案（若已配置）  

**验收**
- 玩家能独立打开图志并看到雾衔/苔行条目状态差异  

### Beat 3 — 第一战（20–30 min）
**目标**：回合/相性/雾幕可读。  
**需求**
- 遭遇轻度失谐敌（棘藤或小虫）  
- 教学：指令、相性提示、雾衔雾幕技能  
- 可设置场地或天气为雾，强化教学  

**验收**
- 玩家不看外部攻略可获胜  
- 复盘提示或 UI 能支持 E5  

### Beat 4 — 露滴蛹结契（30–40 min）
**目标**：培育型结契。  
**需求**
- 发现叶尖蛹  
- 环境条件：遮阴/保湿互动  
- 结契成功入队  
- 提示进化条件与雨将至  

**验收**
- 与静随模板操作手感可区分  

### Beat 5 — 进化高光（40–50 min）
**目标**：成长戏剧。  
**需求**
- 雨天或培育点触发进化  
- Cutscene/Timeline：镜头、音频、伙伴反应  
- 形态变化后技能/定位可感知差异（哪怕小）  

**验收**
- 不可跳过到“无演出直接变模型”的廉价路径作为默认  
- Q3 评审  

### Beat 6 — 棘影事件（50–58 min）
**目标**：伦理与失谐。  
**需求**
- 失谐洼地场景  
- 棘影威胁路线  
- 战斗目标：驱离/净化，不默认捕获  
- 图志记录“痛/失谐”相关层  
- 明确：不是所有相遇都该收入囊中  

**验收**
- 无“一键收服棘影”的默认成功路径  
- E7 访谈可通过  

### Beat 7 — 钩子与展望（58–70+ min）
**目标**：长期路径可见。  
**需求**
- 回声鹿蹄音/远景/可追踪线索任务  
- 静默之心初响（主线旗标）  
- 学院与守护团（或图志）给出 ≥2 条人生路径提示  
- 可存档结束  

**验收**
- 玩家能说出“下一步想做什么”

## 6. 用户故事 + 验收（切片执行集）

格式：故事 → 验收标准（Given/When/Then 精简）

1. **移动**  
   作为旅人，我希望流畅移动，以便探索林海。  
   - 验收：在测试区走完引导路径无卡住；镜头不长期遮挡角色。

2. **交互**  
   作为旅人，我希望靠近可互动物件时看到提示并互动。  
   - 验收：石碑/NPC/迹象点均有提示；按下完成交互。

3. **雾衔静随结契**  
   作为共鸣者，我希望用正确方式让雾衔跟随我。  
   - 验收：正确步骤成功入队；冲撞失败且提示原因；可重试成功。

4. **图志目击**  
   作为研究者，我希望看到精灵后图志出现条目。  
   - 验收：首次目击 C002 外观层解锁。

5. **图志观察**  
   作为研究者，我希望观察后获得生态信息。  
   - 验收：完成观察交互后生态层状态变化。

6. **教学战斗**  
   作为训练师，我希望打赢第一场教学战并理解雾幕。  
   - 验收：使用雾幕或相性建议后可胜；战斗 UI 显示基础因果。

7. **露滴蛹培育结契**  
   作为培育者，我希望通过环境互动结契。  
   - 验收：与静随不同的步骤；成功入队。

8. **进化演出**  
   作为伙伴，我希望看到雨中进化。  
   - 验收：条件满足触发；演出播放；形态与技能更新。

9. **棘影伦理战**  
   作为守护者，我希望驱逐失谐威胁而非默认捕获。  
   - 验收：胜利条件为驱离/净化；无默认收服成功。

10. **羁绊可见**  
    作为伙伴，我希望看到羁绊提升。  
    - 验收：共同战斗或关键事后 UI 数值/阶变化。

11. **发现链**  
    作为探索者，我希望完成隐藏栖息发现。  
    - 验收：2 条链均可完成并给奖励/图志/故事之一。

12. **存档**  
    作为玩家，我希望存档后读档继续。  
    - 验收：Beat3 后存盘，重开恢复位置、队伍、旗标。

13. **路径展望**  
    作为玩家，我希望知道长期能成为谁。  
    - 验收：结尾对话出现 ≥2 路径关键词（学院/图志/守护等）。

14. **数据驱动精灵**  
    作为制作方，我希望不改代码只加数据出现测试种。  
    - 验收：新增 placeholder def+view 后可目击（Dev 验收）。

15. **模式切换稳定**  
    作为玩家，我希望从战斗/结契返回探索状态正确。  
    - 验收：输入、UI、时间暂停状态恢复正确，无软锁。

### 10.1 好的测试
- 测外部行为与规则结果，不测私有字段名  
- Domain 单测：相性伤害、结契状态转移、图鉴解锁、进化条件  
- 手测：60 分钟脚本 Beats 0–7  
- 沙盒：BondingSandbox / BattleSandbox  

### 10.2 必测清单
| 类型 | 用例 |
|---|---|
| 单测 | 静随成功/失败转移 |
| 单测 | 基础伤害 × 相性 |
| 单测 | 目击解锁外观层 |
| 单测 | 进化条件 true/false |
| 手测 | E1–E7 |
| 手测 | 存读档 |
| 手测 | Mode 往返无软锁 |
| 数据 | validate_data 全绿 |

## 12. 依赖与假设

### 依赖
- Unity 工程按架构骨架创建  
- 美术可先 placeholder，但识别点要可辨  
- 文案中文先落地  

### 假设
- 单机本地运行  
- 1 名玩家  
- 16:9 PC  

### 约束
- 性能目标：中端 PC 探索 60fps 为目标（允许 VS 未达优，但不可长期 20fps）  
- 输入：键鼠+手柄

## 13. 反需求（体验禁止）

1. 结契唯一进度条小游戏无理解语义  
2. 战斗纯平A无信息  
3. 教程超过 10 分钟纯站桩对话  
4. 空跑 >90 秒无引导  
5. 棘影被当成普通抓宠教程  
6. 进化无演出  
7. 图志只有数值  
8. 为赶工破坏 Domain 纯净与数据驱动
