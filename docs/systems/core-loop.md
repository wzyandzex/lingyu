# 核心循环（Core Loop）v0.1

> 配套支柱见 `docs/product/pillars.md`  
> 目标：把“活世界 + 关系 + 发现”落成可重复的快乐节奏

---

## 1. 循环总图

```text
探索世界 → 观察生态 → 共鸣结契 → 共同经历（养成）
    ↑                                         ↓
长期身份/图志 ← 事件与叙事 ← 战斗/试炼 ← 队伍表达
```

---

## 2. 四级时间循环

### 2.1 分钟级（30–180 秒）
- 移动到兴趣点
- 发现迹象（足迹、叫声、雾气异色）
- 小交互：观察 / 采集 / 对话 / 遭遇

**爽点**：频繁微发现，不空跑。

### 2.2 小时级（10–20 分钟）
- 完成一次结契或一场有决策的战斗
- 解锁一条图志条目层
- 推进一个小目标（送达、净化、研究、护送）

**爽点**：关系或能力出现可感知变化。

### 2.3 天级（单次登录 45–90 分钟）
- 打通区域小节
- 完成进化/事件节点
- 身份声望小提升
- 设定下一次旅行目标

**爽点**：地图认知扩张 + “我的队伍更像我了”。

### 2.4 赛季/篇章级
- 世界状态变化（迁徙、失谐潮、节日）
- 传说追踪与大型事件
- 新区域/新形态/新研究课题

**爽点**：世界仍活着，不是静态地图包。

## 3. 四大核心系统规格（v0.1）

### 3.1 结契（Bonding）

**流程**
1. **目击**：剪影/声音进入图志外观层  
2. **观察**：记录习性，解锁生态层线索  
3. **试探**：按行为谱系互动（投喂、静随、节奏、帮助）  
4. **共鸣判定**：成功率 = 理解度 + 环境匹配 + 道具辅助 − 威胁动作  
5. **契约成立/失败**：失败写明原因；成功进入队伍/牧场

**原则**
- 不同精灵不同仪式
- 禁止唯一解为“丢球”
- 研究道具提高信息，不直接秒杀过程

### 3.2 养成与羁绊（Growth & Affinity）

**资源**
- 经验/训练：战斗与练习
- 羁绊：共历、正确互动、叙事选择
- 性格偏移：长期行为统计

**产出**
- 技能表现修饰（不是只有数值）
- 字段技能/旅技解锁
- 形态稳定度与特殊事件

**反琐碎**
- 不做每日浇水强迫
- 关键互动合并进旅行自然行为

### 3.3 战斗（Battle）

**基底**：回合制 + 场地/天气 + 轻站位（前/后排）

**元素**
- 澄相相性
- 状态（湿润、灼伤、紊乱、共生等）
- 场地（雾林、潮坪、晶脉…）
- 组合技（特定配对）
- 羁绊技（关系阈值）

**目标时长**
- 普通战：1–3 分钟
- 精英/事件战：3–6 分钟
- 教学战必须 3 回合内教完一个因果

### 3.4 探索（Exploration）

**兴趣点类型**
- 生态位点、隐藏栖息、民谣碑、失谐裂口、工匠/研究者营地

**动态层**
- 昼夜、天气、季节、区域事件改遭遇与路径

**奖励**
- 图志层、材料、捷径、故事，不只是金币

## 4. 系统依赖（实现顺序建议）

```text
1. 移动/镜头/交互
2. 区域流送与兴趣点
3. 精灵实体 + 图志目击
4. 观察与结契
5. 队伍与基础战斗
6. 羁绊与养成反馈
7. 进化演出
8. 世界事件与主线旗标
```

---

## 5. 数值框架结构（仅结构，非终表）

| 域 | 结构 |
|---|---|
| 精灵成长 | 等级 1–N；阶段阈值；个体差异低波动 |
| 羁绊 | 0–5 阶；每阶解锁表达而非纯攻防 |
| 战斗力 | 基础六维 + 澄相修正 + 场地修正 + 羁绊修饰 |
| 经济 | 材料主、货币辅；避免一切可买 |
| 难度 | 区域推荐共鸣深度；允许非最优通过 |

原则：**可读因果 > 复杂公式崇拜**。

---

## 6. 失败与防劝退

- 战斗失败：掉落“研究线索/创伤回忆”，不毁队伍
- 结契失败：增加图志信息
- 迷路：地图有环境导航（光、声、伙伴提示）
- 强度卡关：提供观察对策，不只刷级

---

## 7. 验收

- [ ] 任意 15 分钟片段能完整走通“发现→互动→反馈”
- [ ] 羁绊变化玩家可感知
- [ ] 战斗结果可用“我看懂了”复盘
- [ ] 探索奖励不全是同质材料
